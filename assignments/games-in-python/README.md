
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman game in Python that practices string handling, loops, conditionals, and user interaction.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description

Create a Hangman game where the player guesses letters to reveal a hidden word before running out of attempts.

#### Requirements

Completed program should:

- Randomly select a word from a predefined list
- Accept letter guesses from the player
- Display the current word progress using placeholders like `_ _ a _ _`
- Track incorrect guesses and remaining attempts
- Show a win message when the word is guessed
- Show a lose message when the player runs out of attempts
- Prevent repeated guesses from counting as extra wrong attempts
